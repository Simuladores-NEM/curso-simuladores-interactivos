# curso-simuladores-interactivos
Repositorio institucional de simuladores interactivos educativos para docentes de matemáticas.
# Curso: Creación de Simuladores Interactivos

Este repositorio contiene los materiales y proyectos del curso institucional para docentes de matemáticas.

---

## 📑 Índice de Módulos

1. [Módulo 1: Fundamentos](Modulo1_Fundamentos/)
   - Teoría pedagógica (Ausubel, Brousseau, Polya)
   - Introducción a HTML y CSS
   - Primer simulador básico

2. [Módulo 2: Diseño](Modulo2_Diseño/)
   - Estética con CSS
   - Interactividad con JavaScript
   - Arrays para guardar datos

3. [Módulo 3: Evaluación](Modulo3_Evaluacion/)
   - Exportación de evidencias con jsPDF
   - Bitácora digital del curso
   - Ejemplos de rúbricas

4. [Módulo 4: Publicación](Modulo4_Publicacion/)
   - Publicar proyectos en GitHub Pages
   - Documentación institucional
   - Presentación final del simulador
